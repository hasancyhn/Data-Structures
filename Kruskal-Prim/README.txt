This code is a program that calculates the shortest path based on user input using Prim's and Kruskal's algorithms. Its functionalities are as follows:

1- The code defines the "Edge" class, which represents an edge with source node, destination node, and weight.
2- A comparison function is defined to compare edges.
3- The "graph" class is used to represent the graph.
4- It takes the number of vertices and edges of the graph, and then it prompts the user to input the edges and stores them.
5- The "unionfind" function implements the Kruskal's algorithm and returns the Minimum Spanning Tree (MST) edges.
6- The "prims" function implements the Prim's algorithm and calculates the cost of the Minimum Spanning Tree (MST).
7- The "main" function presents a menu to the user and runs the corresponding algorithm based on the user's choice.
This code is a program that implements Minimum Spanning Tree (MST) calculation algorithms and provides user interaction.
