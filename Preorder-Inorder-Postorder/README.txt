This project creates a binary search tree data structure and performs various operations on it. The functions are as follows:

kokDugumOlustur: Creates a new root node and assigns the data to it.
elemanEkle: Inserts a new node under the given root node. The data is placed on the appropriate side of the node (left or right).
enKucukVeri: Finds the minimum value in the tree and returns it.
enBuyukVeri: Finds the maximum value in the tree and returns it.
elemanSil: Deletes a given value from the tree.
inorder: Traverses the tree in inorder (left-root-right) and prints the nodes to the console.
dugumSayisi: Counts the number of nodes in the tree and returns the count.
yukseklik: Calculates the height of the tree and returns it.
fullBinaryKontrol: Checks if the tree is a full binary tree.
completeBinaryKontrol: Checks if the tree is a complete binary tree.
In the main function, a menu is presented to the user, and the corresponding operations are performed based on the user's selection. The user can add elements, delete nodes, print the tree in inorder traversal, and access other information. The program continues to run until the user chooses to exit.

The purpose of this project is to demonstrate the basic operations and some features of a binary search tree data structure.

Bu proje, bir ikili arama ağacı (binary search tree) yapısını oluşturmayı ve üzerinde çeşitli işlemler gerçekleştirmeyi sağlar. İşlevler şu şekildedir:

kokDugumOlustur: Yeni bir kök düğüm oluşturur ve veriyi içine yerleştirir.
elemanEkle: Verilen bir kök düğümün altına yeni bir düğüm ekler. Veri, düğümün uygun tarafına (sol veya sağ) yerleştirilir.
enKucukVeri: Ağacın en küçük verisini bulur ve geri döndürür.
enBuyukVeri: Ağacın en büyük verisini bulur ve geri döndürür.
elemanSil: Verilen bir değeri ağaçtan siler.
inorder: Ağacı inorder (soldan kök, sağa) şeklinde dolaşarak düğümleri ekrana yazdırır.
dugumSayisi: Ağaçtaki düğüm sayısını hesaplar ve geri döndürür.
yukseklik: Ağacın yüksekliğini hesaplar ve geri döndürür.
fullBinaryKontrol: Ağacın tam ikili ağaç olup olmadığını kontrol eder.
completeBinaryKontrol: Ağacın tamamlanmış ikili ağaç olup olmadığını kontrol eder.
Ana fonksiyonda ise kullanıcıya bir menü sunulur ve kullanıcının seçimine göre ilgili işlemler yapılır. Kullanıcı eklemeler yapabilir, düğümler silebilir, ağacı inorder şekilde sıralayabilir ve diğer bilgilere erişebilir. Menüden çıkış yapılana kadar program çalışmaya devam eder.

Bu projenin amacı, ikili arama ağacı yapısının temel işlevlerini ve bazı özelliklerini göstermektir.
