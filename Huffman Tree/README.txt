This code aims to perform compression of a given word using Huffman coding and calculate code statistics. Here is a brief summary of how it works:

1- The code takes input from the user, which is a word, and calculates its length.
2- If the word is empty, the program stops.
3- The frequency of each symbol is calculated, and the symbols are sorted in descending order based on their frequencies.
4- To construct the Huffman tree, the symbols are converted into nodes.
5- Huffman coding is applied to generate codes for each symbol.
6- Code statistics are calculated and printed to the screen.
7- Using the Encoder class, the word is compressed into its encoded form.
8- The compressed message is printed to the screen.
The purpose of this code is to utilize Huffman coding for word compression and to calculate code statistics.
